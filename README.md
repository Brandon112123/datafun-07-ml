## Project Overview

This project introduces core machine learning concepts through the application of simple linear regression. The objective is to analyze historical temperature data and forecast future trends using both statistical and machine learning approaches.

The project uses two historical temperature datasets (1895–2018) to analyze long-term trends and compare regression modeling approaches.

---

## Datasets

This project uses the following datasets:

### 1. NYC Average January High Temperatures  
File: `ave_hi_nyc_jan_1895-2018.csv`

Columns:
- `Date` (Year)
- `Value` (Average January high temperature in Fahrenheit)
- `Anomaly` (Deviation from long-term average)

---

### 2. LA Average January High Temperatures  
File: `ave_hi_la_jan_1895-2018.csv`

Columns:
- `Date` (Year)
- `Value` (Average January high temperature in Fahrenheit)
- `Anomaly` (Deviation from long-term average)
